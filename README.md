# Verboten

A voice-based demo game where the human player speaks to describe a word, while the AI model listens
and guesses the word.

## How to play

- Install the [Go](https://go.dev) programming language
- In the root folder of the repo, launch the command `go run ./cmd/web`
- Open your browser at http://localhost:8080/

## Limitations

Currently works only on Chrome or on Android.
